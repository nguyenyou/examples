Resize Container Demo

Drag the resize handle in the bottom-right corner of each container to resize.

1. Resize Width Only
This container can only be resized horizontally. Drag the resize handle to change the width. The height remains fixed.
2. Resize Height Only
This container can only be resized vertically. Drag the resize handle to change the height. The width remains fixed.
3. Resize Both Width and Height
This container can be resized in both directions. Drag the resize handle to change both width and height. You have full control over the container dimensions.
